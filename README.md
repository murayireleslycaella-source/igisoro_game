## 📌 Project Description

This project is a digital version of the Igisoro game. It follows the official rules including:
- Board setup (4 rows × 8 pits)
- Turn-based gameplay
- Sowing and relay moves
- Capture mechanics
- Pass and retreat system
- Win condition logic

The game is designed to be interactive, simple to play, and visually modern.

---

## 🎮 How to Play

### 1. Start the Game
- Player 1 starts first.

### 2. Select a Pit
- Click a pit in your territory.
- You cannot choose a pit with only 1 seed.

### 3. Sowing
- Seeds are distributed one by one across your territory.

### 4. Relay Rule
- If the last seed lands in a non-empty pit, you continue moving.

### 5. Capture
- If conditions are met, you may capture opponent seeds.
- You can choose to pass instead.

### 6. Pass and Retreat
- If you pass, the opponent may perform a retreat move.

### 7. Direction Change
- During capture, you may choose clockwise or counterclockwise movement.

### 8. Winning
- A player wins when the opponent has no valid moves left.
